# Geo Collectivités Explorer

Mini application web SIG réalisée dans le cadre de ma montée en compétence en géomatique et développement cartographique web.

## Objectif
Permettre la recherche et la visualisation de communes françaises sur une carte interactive, avec affichage d’informations administratives et de géométries GeoJSON.

## Fonctionnalités
- Recherche de communes
- Affichage cartographique interactif
- Visualisation GeoJSON
- Popup et zoom sur entité
- Panneau d’informations administratives

## Stack
- Next.js
- TypeScript
- React-Leaflet / Leaflet
- Tailwind CSS

## Sources de données
- geo.api.gouv.fr
- OpenStreetMap tiles

## Notions SIG mobilisées
- GeoJSON
- données vecteur
- contour administratif
- carte web interactive
